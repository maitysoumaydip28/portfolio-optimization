\# Portfolio Optimization Model



\## Objective

To optimize a stock portfolio for maximum return and minimum risk.



\## Tools Used

Python, Pandas, NumPy, Matplotlib, yFinance



\## Methodology

\- Collected stock data from Yahoo Finance

\- Calculated daily returns

\- Generated random portfolios

\- Calculated risk and return

\- Plotted Efficient Frontier



\## Output

\- Efficient Frontier Graph

\- Optimal portfolio risk \& return



\## Conclusion

This model helps investors choose the best portfolio based on risk-return trade-off.
